# rn-wifi-p2p

React Native module for Android P2P communication using Wi-Fi Direct.

## Installation

```sh
npm install rn-wifi-p2p
```

## Usage


```js
import { multiply } from 'rn-wifi-p2p';

// ...

const result = multiply(3, 7);
```
